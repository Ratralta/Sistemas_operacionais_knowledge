#  O que é :
* É uma maneira do windons utilizar distribuições linux 
# Como usar :
* no [PowerShell](PowerShell.md) do Windows , você irá digitar *wsl*, assim você abre o [terminal do linux](Terminal_do_Linux.md), dentro dele você está no linux, podendo fazer as coisas que você quiser através dele.
## Comandos básicos :
* wsl --list --> Serve para ver quais distribuições linux estão instaladas no computador. 
 
 # Instalar e Desinstalar  :
* Para instalar, é só usar o comando "wsl --install", de maneira padrão irá abaixar o "wsl" junto da distro linux "Ubuntu".
* [Informações sobre instalação](https://learn.microsoft.com/pt-br/windows/wsl/install)
* Para desinstalar o "wsl" , é só usar o comando "wsl --uninstall" no PowerShell.

# O que é :
* É um meio de armazenamento (como pendrive e CD) que possui arquivos necessários para a instalação/inicialização de um sistema operacional. 
* Resumidamente, é um pendrive com um arquivo [.iso](iso) nele capaz de inicializar um sistema operacional. 

# Como criar disco de boot :
* Para criar um disco de boot, você precisa de um software, um deles é o [Rufus](https://rufus.ie/pt_BR/)
* Com o Rufus, coloque seu dispositivo de armazenamento e selecione ele.  
* Então com sua .iso abaixada, coloque ela e aperte pronto.
* O DISPOSITIVO DE ARMAZENAMENTO SERÁ FORMATADO DURANTE O PROCESSO.

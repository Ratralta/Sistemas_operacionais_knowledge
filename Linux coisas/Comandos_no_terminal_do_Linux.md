# Siglas do terminal :
* **^** : É a tecla "Control". 
	* EX : ^C  -- > Control + C
* **M** : É a tecla "Alt".
	*  EX : M-D --> Alt + D
# Comandos básicos linux : 
* **ls** --> Serve para ver os arquivos e pastas que existem em seu [diretor](Diretor.md) atual. 
* **touch arquivo.txt** --> Cria um arquivo no *diretor atual*. 
* **nano arquivo.txt** --> Entra em um arquivo de texto através do [terminal](Terminal_do_Linux.md). "nano" é um editor de texto básico do linux. 
* **cat arquivo.txt** --> Mostra o conteúdo do arquivo selecionado.

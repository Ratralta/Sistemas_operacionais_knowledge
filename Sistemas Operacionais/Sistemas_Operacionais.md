# Informações : 
* Todos os sistemas operacionais precisam/ocupam :
	* Ocupa um espaço no disco rígido.
	* Um espaço na memoria.

* Arquivos "**.iso**", são sistemas operacionais compactados.  

# Como abaixar sistemas operacionais : 
## Windos : 
### Windows 7 :
* Com sua iso do windons.
* Na parte de abaixar, você poderá ver a memoria de seu HD separado em [[Partições]], uma será para seu sistema e outro para o uso normal , é recomendado criar mais uma partição para beckup caso precise. Depois aperte em avançar 
* Pronto ai deu certo a instalação. 
## Linux :
### Ubuntu : 
* Com sua iso do Ubuntu.
* Faça as seleções que você deseja utilizar em seu linux.
